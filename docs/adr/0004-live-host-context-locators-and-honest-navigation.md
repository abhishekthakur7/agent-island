# Live Host Context locators and honest navigation

Agent Island records Host Context associations separately from Product-owned Agent Session identity and activates only a currently revalidated, Host-specific locator. This favors an explicit historical/recreated context and a reported lower Jump Back level over a plausible match from titles, paths, geometry, Spaces, or accessibility data, because false navigation can direct a person to the wrong coding work.

## Consequences

- Host restart, endpoint loss, permission revocation, and unresolved re-creation invalidate navigation capability without changing Agent Session lifecycle state.
- Every Jump Back reports its achieved `exactSurface`, `exactTab`, `workspaceOrFile`, `windowBestEffort`, `appOnly`, or `unavailable` level; navigation never confers action authority.
- A Space, full-screen placement, and non-unique Accessibility window remain diagnostic/presentation evidence, never a durable target.
