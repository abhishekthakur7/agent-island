# Triage Labels

The skills use five canonical triage roles. This file maps them to the Linear labels for this repository.

| Canonical role | Linear label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate the issue. |
| `needs-info` | `needs-info` | Waiting on the reporter for more information. |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for an agent. |
| `ready-for-human` | `ready-for-human` | Requires human implementation or participation. |
| `wontfix` | `wontfix` | The issue will not be actioned. |

When a skill refers to a canonical role, apply the corresponding Linear label.
