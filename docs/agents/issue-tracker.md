# Issue Tracker: Linear

Issues for this repository are tracked in Linear, in project `agent-island`.

## Workflow

- Create and update work items in the `agent-island` project using the configured Linear team and its default workflow.
- Treat Linear as the source of truth for issue status, ownership, discussion, and completion.
- Reference the Linear issue identifier and URL when linking work from repository documentation, commits, or pull requests.
- Use the project's default issue type, priority, and status unless a task explicitly specifies otherwise.

## Triage

Apply the label mapping in `docs/agents/triage-labels.md` when evaluating incoming work. Use the team's normal Linear statuses to reflect progress; labels describe triage state and do not replace status.
